Time Slot Widget

User Story:

Ask a marketing manager I would like a widget created that is injected into the page that displays time slots for my booking application. I believe that by putting this widget on our location pages will drive a higher conversion rate when showing time slots higher up in the users search funnel. I need a developer to create a stand alone script that can be added to the page to drive this functionality.

Instructions

  • Build the widget below and inject it into this page in the #addContentHere selector found on this page
  • Feel free to use any CSS or JS that is loaded into the DOM of this page
  • If additonal CSS is needed add a CSS file upon delivery
  • Use the data from the API Link found in the development requirements to populate the timeslots
  • For the time slots use the first eleven time slots from the API
  • For the "What we treat" and "How much will it cost" these are links just make them link to this page
  • Make all the time slot buttons as well as the more button link to this page when clicked
  • On devices with a screen larger than 991px the time slot buttons must be in columns of four but its up to the developer for tablet and mobile devices
  • Once built and tested inject into this page and take a screen shot
  • Hand off the JavaScript in a JS file along with any additional assets as well a screen shot of the wiget injected into the page
  • Email results to drew@wheelhousedmg.com
